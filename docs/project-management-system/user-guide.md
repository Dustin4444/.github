# User Guide

## Dashboard Overview

The Dashboard is your command center for project management. It displays:

- **Statistics Cards**: Shows count of active projects, in-progress tasks, and completed tasks
- **Projects Section**: All your active projects with quick access
- **Recent Tasks**: Latest tasks across all projects

### Navigating the Dashboard

1. Click "Dashboard" in the navbar to return to the main view
2. Use "New Project" and "New Task" buttons to create new items
3. Click on any project or task card to view details

## Managing Projects

### Creating a Project

1. Click the "New Project" button from the dashboard
2. Enter a **Project Name** (required)
3. Add an optional **Description**
4. Click "Create Project"

Your new project will appear on the dashboard immediately.

### Viewing Project Details

1. Click on a project card from the dashboard
2. View all tasks associated with this project
3. Create new tasks directly from the project view

### Project Fields

- **Name**: The title of your project
- **Description**: Brief overview of the project's purpose
- **Status**: Projects can be active or archived
- **Owner**: The user who created the project
- **Created**: Timestamp of project creation

## Managing Tasks

### Creating a Task

1. Click "New Task" from the dashboard or project view
2. **Select a Project** (required)
3. Enter a **Task Title** (required)
4. Add an optional **Description**
5. Set the **Priority**: Low, Medium, or High
6. Choose a **Due Date** (optional)
7. Click "Create Task"

### Task Properties

- **Title**: What needs to be done
- **Description**: Detailed explanation of the task
- **Status**: Todo → In Progress → Done
- **Priority**: Low, Medium, High
- **Assigned To**: Team member responsible
- **Due Date**: Target completion date
- **Created By**: Who created the task

### Updating Task Status

1. Open a task by clicking on it
2. Use the "Status" dropdown to change between:
   - **Todo**: Not started yet
   - **In Progress**: Currently being worked on
   - **Done**: Completed
3. Changes save automatically

### Updating Task Priority

1. Open a task details page
2. Use the "Priority" dropdown to select:
   - **Low**: Can be done later
   - **Medium**: Should be done soon
   - **High**: Urgent, needs immediate attention
3. Updates apply instantly

## Task Cards

Each task displays:

- ✓ **Completion Icon**: Shows if task is done
- **Title**: Task name
- **Description**: Brief summary
- **Status Badge**: Current status with color coding
- **Priority Indicator**: Visual priority level
- **Due Date**: When it's due (if set)

### Task Card Colors

- **Gray**: Todo status
- **Blue**: In Progress status
- **Green**: Done status

## Filtering and Searching

### Filter Tasks by Status

Use the status dropdown in the task view to show only:
- All tasks
- Todo only
- In Progress only
- Done only

### View Tasks by Project

Navigate to a specific project to see only its tasks.

## Troubleshooting

### Task not saving
- Check that all required fields are filled
- Ensure your internet connection is active
- Refresh the page and try again

### Project not appearing
- Clear your browser cache
- Verify the project was created (check dashboard)
- Try logging out and back in

### Can't update task status
- Make sure the task exists
- Verify you have permission to edit
- Try refreshing the page

For more help, contact your system administrator.
